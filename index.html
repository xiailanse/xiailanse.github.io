关于Markdown

Markdown是一种用来写作的轻量级「标记语言」，它用简洁的语法代替排版，而不像一般我们用的字处理软件 Word 或 Pages 有大量的排版、字体设置。它使我们专心于码字，用「标记」语法，来代替常见的排版格式。例如此文从内容到格式，甚至插图，键盘就可以通通搞定了。

我也要好好学习下了markdown语法

这里写一些比较常用到的一些语法

标题是每篇文章都需要也是最常用的格式，在 Markdown 中，如果一段文字被定义为标题，只要在这段文字前加 # 号即可。

以此类推，总共六级标题，建议在井号后加一个空格，这是最标准的 Markdown 语法。

想给句话或者词加上超链接的时候，只需要加上两个括号就行了。

[文字](https://链接地址.html)

熟悉 HTML 的人肯定知道有序列表与无序列表的区别，在 Markdown 下，列表的显示只需要在文字前加上 - 或 * 即可变为无序列表，有序列表则直接在文字前加1. 2. 3. 符号要和文字之间加上一个字符的空格。

###无序列表

  • 2
  • 3
  • 4
  • 5
    ###有序列表
  1. 第一个
  2. 第二个
  3. 第三个

插入链接与插入图片的语法很像，区别在一个 !号

图片为：![](){ImgCap}{/ImgCap}

链接为：[]()

Markdown 的粗体和斜体也非常简单，用两个 * 包含一段文本就是粗体的语法，用一个 * 包含一段文本就是斜体的语法。

**粗体**
*斜体*

如果你是个程序猿，需要在文章里优雅的引用代码框，在 Markdown下实现也非常简单，只需要用两个 ` 把中间的代码包裹起来
