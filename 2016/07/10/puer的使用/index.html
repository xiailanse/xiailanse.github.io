puer 的使用

###puer简介
puer是个可以实时编辑刷新的服务器。

  • 提供一个当前或指定路径的静态服务器
  • 所有浏览器的实时刷新：编辑css实时更新(update)页面样式，其它文件则重载(reload)页面
  • 提供简单熟悉的mock请求的配置功能，并且配置也是自动更新。
  • 可用作代理服务器，调试开发既有服务器的页面，可与mock功能配合使用
  • 集成了weinre，并提供二维码地址，方便移动端的调试
  • 可以作为connect中间件使用(前提是后端为nodejs，否则请使用代理模式)

然而以上的大部分功能我都不会用，大概我现在只是用来偷懒，以后修改文件代码后不用老是按 F5刷新页面了。啊哈哈哈!

###让我们开始使用把！

####安装
首先我们要安装了nodejs。

没有安装的点我下载安装

然后使用npm全局安装puer

npm install puer -g

想要帮助可以输入puer -h，进行查看。

####使用
大部分情况下我们只用 两句 就可以了！

cd /我的/路径

puer
